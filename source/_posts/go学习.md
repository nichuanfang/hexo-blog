# go 学习 技巧

## 1. 语言特性

go 的优点:

- 语法简单
- 编译快
- 并发 编程
- 内存管理

## 2. 语法

### 2.1. 基本语法

#### 2.1.1. 变量

```go
// 声明变量
var a int
var b string
var c bool
var d  float32
var e float64
var f byte
var g rune
var h complex64
var i complex128
var j []int
var k [10]int
var l map[string]int
var m func(a int) int
var n *int
```

#### 2.1.2. 常量

```go
// 声明常量
const a int = 10
const b string = "hello"
const c bool = true
```
